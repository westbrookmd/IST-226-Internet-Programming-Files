My Portfolio: Graded Practice Files from IST226

Live Link: Web Server Setup

Creating a web server account and submitting a test page to demonstrate our knowledge and completion of this step.


Live Link: Chapter 1

A basic introduction to HTML5. Covered learning about networks, web pages, web servers, the history of HTML, elements, file creation, character encoding, and basic grouping via lists. Additionally covered using classes and id attributes to better organize pages.


Live Link: Chapter 2

This chapter introduced style sheets through Cascading Style Sheets (CSS). This chapter discussed using different font sizes and learning about scaling to the appropriate view port. A brief introduction to padding, margins, classes, and more.


Live Link: Chapter 3

This chapter covered page layouts. Fixed, fluid, elastic, and grid layouts were covered. Positioning was emphasized to explain how to not corner your design into just using grids.


Live Link: Chapter 4

This chapter covered graphic design using CSS. This was a lot of information to cover. Most of the difficult parts for me were concerned with images, whether it be changing background images or editing images with CSS transforms or filters. Gradients were my favorite part of this chapter.


Live Link: Chapter 5

This chapter was focused on mobile web design. This introduced responsive design (which I tried my hardest to design this portfolio as a responsive layout). Flexible boxes, flexible items, and flex box layouts were the hardest parts for me to grasp.


Live Link: Chapter 6

This chapter was about tables and columns. Table borders, spans for rows and columns, and using groups were the highlights that I really paid close attention to. Column layouts were hard to grasp at first, but I really began to understand them from this graded practice file.


Live Link: Chapter 7

This chapter and graded practice was focused on web forms. I never knew how many different types of input there were before this chapter. The input type that I feel like I will use the most (that I didn't know about before) is the "tel" input type. Being able to use this type to give mobile users a number pad by default is great.


Live Link: Chapter 8

This chapter and graded practice was all about multimedia. I really enjoyed this graded practice because it had an excellent video that encouraged transitions, key frames, and a cue text. Additionally, learning about how pages with transitions are rendered was really interesting.


Live Link: Chapter 9

This chapter and graded practice was all about JavaScript. JavaScript was not something I expected to learn in this class, but it very quickly became something that I sought out and read about since it is still very new to me. Learning how to dynamically change HTML code via JavaScript is something that is amazing once you get the hang of it. I utilized event handlers in this page for each of the chapters of my portfolio thanks to this chapter.


Live Link: Chapter 10

This chapter and graded practice described how arrays, loops, and conditional statements work in JavaScript. This chapter made me realize how different JavaScript really is compared to Java, C#, Python, and basically any other language I have used. Not having the strict typing that C# and Java offer is really aggravating. I expect to learn TypeScript in the near-future to work around this feature. I like how mapping and filtering was described very early in this chapter. Most other books that I have read wait so long before introducing these concepts but I feel as if they are critical to learn early.
